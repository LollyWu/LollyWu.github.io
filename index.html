GitHub Pages 与响应式网页设计入门

什么是 GitHub Pages?

GitHub Pages 是一项静态站点托管服务，它直接从 GitHub 上的仓库获取 HTML、CSS 和 JavaScript 文件并发布网站。

// 优势

  • 完全免费
  • 自定义域名支持
  • 自动化部署 (GitHub Actions)

快速上手步骤

STEP 01

创建仓库

在 GitHub 上新建一个名为 username.github.io 的仓库（将 username 替换为你的 GitHub 用户名）。

STEP 02

上传文件

将你的 index.html 提交并推送（Push）到该仓库的 main 分支。

STEP 03

开启服务

进入仓库的 Settings > Pages，确保 Source 设置为 "Deploy from a branch" 并选择主分支。

STEP 04

即刻访问

几分钟后，你的网站将通过 https://username.github.io 公开发布。

响应式布局 (Responsive Design)

为了让网页在手机、平板和桌面端都有良好表现，我们使用了以下关键技术：

  • Viewport Meta: 告诉浏览器按照设备的宽度渲染。
  • CSS Grid / Flexbox: 动态调整元素排列。
  • Media Queries: 针对不同屏幕尺寸应用不同样式。
